# ANDREA DELA CRUZ POSTMAN API_CALL DEMONSTRATION

## About the Project

In this project, I created and tested API calls using Postman to understand how APIs work and how different HTTP methods function. This project demonstrates the use of GET, POST, PUT, PATCH, and DELETE requests together with their corresponding responses using Laravel and Postman.

# API Methods Demonstrated

## GET Request

Retrieves all quiz records from the database.

## POST Request

Creates a new quiz record in the database.

## PUT Request

Updates an existing quiz record.

## PATCH Request

Partially updates an existing record.

## DELETE Request

Deletes a quiz record from the database.

# Setup Instructions

1. Install dependencies

```bash
composer install
```

2. Configure the database in the `.env` file

3. Run migrations

```bash
php artisan migrate
```

4. Start the Laravel development server

```bash
php artisan serve
```
